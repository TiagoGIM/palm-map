# CLAUDE.md

Este projeto usa AGENTS.md em cada pasta como fonte de contexto e regras de execução.

## Como se orientar

- Antes de qualquer tarefa, leia o AGENTS.md e README.md do diretório relevante
- Prefira contexto local (pasta atual) antes de subir na hierarquia
- Não explore o repositório inteiro — leia apenas o necessário para a tarefa
- As regras globais estão em /AGENTS.md

## Ferramentas e comportamento

- Use `pnpm` para instalar dependências e rodar scripts
- Prefira editar arquivos existentes a criar novos
- Atualize a documentação (README.md / AGENTS.md) quando contratos ou fluxos mudarem
- Mudanças estruturais: leia README.md e docs/product/mvp.md antes de agir
